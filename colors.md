## Color Usage Guidelines

### Accessibility
Always ensure sufficient contrast between text and background colors to meet WCAG 2.1 AA standards (minimum contrast ratio of 4.5:1 for normal text).

### Combinations
- Use Vermont Green with white or light gray backgrounds
- Use Data Blue for data visualizations and charts
- Pair primary colors with neutrals for balance
- Use secondary colors sparingly for accents

### Data Visualization
When creating charts and graphs:
1. Start with Data Blue as the primary color
2. Use Vermont Green as a complementary color
3. Add secondary colors for additional data series
4. Ensure colors are distinguishable for colorblind users
